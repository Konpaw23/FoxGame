# Unity Game Project

Unity game created during classes at Gdańsk University of Technology.

## About the Project

This project was created during university classes and focuses on learning core game development concepts such as:

* gameplay mechanics
* scripting in C#
* scene management
* basic UI systems

## Features

* Playable game prototype
* Custom game logic
* Interactive elements
* Built using Unity Engine

## Technologies

* Unity
* C#
* Visual Studio
